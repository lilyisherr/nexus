About Nexus

A YouTube bot and analytics tool built by someone who got tired of the existing options.

What is Nexus?

Nexus started because I couldn't find a decent YouTube stream management tool that wasn't either overpriced, half-broken, or clearly designed for Twitch first with YouTube support tacked on as an afterthought. So I built one.

It's a bot and analytics platform made specifically for YouTube. Live chat moderation, custom commands, stream analytics, per-channel settings -- all in one dashboard. You sign in with Google, connect your channel, and it just works. No third-party accounts, no complicated setup, no monthly fees during beta.

The whole thing runs on Flask and Python, talks to the YouTube Data API, and uses polling to read live chat in near real-time. It's not perfect yet, but it's been in active development for close to a year and it's getting better every week.

What drives Nexus

YouTube-First

Built around YouTube's API from day one. Not a Twitch tool with a YouTube checkbox. The polling, the OAuth flow, the chat handling -- all of it is designed for how YouTube actually works.

Privacy by Default

Google OAuth handles authentication, so Nexus never touches your password. I store what's needed to make the bot work and nothing more. No trackers, no data selling.

Feedback-Driven

Features get built because someone actually asked for them. The roadmap is shaped by the people using Nexus, not by whatever sounds good on a landing page.

No Bloat

If a feature doesn't make streaming or managing your channel easier, it doesn't get built. The goal is a tool that's fast, focused, and actually useful.

How It Started

April 2025
Project Started
Initial research into YouTube's Data API and Live Streaming API. Set up the Flask project structure, started figuring out OAuth scopes and quota limits. Mostly just reading docs and prototyping API calls.
June 2025
First Prototype
Got basic YouTube API integration working. Could pull channel data, list live broadcasts, and read chat messages. Everything was held together with duct tape and print statements, but it proved the concept was viable.
August 2025
Bot System Working
Live chat polling was stable enough to actually use. The bot could join a stream, read messages, and respond to basic commands. Spent a lot of time tuning the polling interval to stay under YouTube's quota limits without missing messages.
October 2025
Moderation Filters
Added the moderation system -- spam detection, excessive caps filtering, link filtering, and a configurable blocked words list. Each filter can be toggled and configured per channel. This was the first feature that made Nexus feel like a real product.
December 2025
Analytics and Per-Channel Settings
Rebuilt the analytics dashboard from scratch after the first version turned out to be a mess. Added per-channel settings so creators managing multiple channels could configure each one independently. Migrated from SQLite to Postgres for production.
February 2026
Custom Commands and Webhooks
Shipped custom commands with cooldown support, so creators can set up their own chat commands without writing code. Also added webhook integrations for external notifications. The command system went through three rewrites before it felt right.
March 2026
Public Beta Launch
Nexus opened up to the public. Core features are stable: bot management, moderation filters, custom commands, analytics, and per-channel configuration. Still a lot to build, but the foundation is solid and people can finally try it out.
