Help Center

Everything you need to know about setting up and using Nexus Bot

Getting Started Dashboard Moderation Welcome System Logging Nexus Integration Troubleshooting

Getting Started

How do I add Nexus Bot to my server?
Sign in with Discord on this site, go to your Dashboard, and select the server you want to configure. You need the Manage Server permission in that Discord server to see it listed. The bot needs to be invited to your server separately using the invite link.
What permissions does the bot need?
The bot needs Administrator permission to set up channels, roles, and manage moderation. If you want more granular control, the minimum permissions are: Manage Roles, Manage Channels, Kick Members, Ban Members, Manage Messages, View Audit Log, Send Messages, Embed Links, Read Message History, and Manage Nicknames.
How do I set up my server with one command?
Use the /setup-server slash command in your Discord server. This creates all roles (Admin, Moderator, Support Team, etc.), channels (organized into categories), voice channels, posts rules, welcome messages, FAQ, and role picker. Only admins can run this command.
Can I undo the server setup?
Yes. Use /reset-server to remove all bot-created channels and roles. You will need to type CONFIRM within 30 seconds to proceed. This is irreversible.

Dashboard

Why don't I see my server on the dashboard?
You need the Manage Server permission or be the owner of the server to see it listed. If you recently changed permissions, try logging out and back in to refresh your server list.
What does each setting do?
Prefix: The character that triggers text commands (default: !).
Welcome Messages: Sends a message when someone joins. Use {user} to mention them.
Auto Role: Automatically gives new members a role on join.
Anti-Spam: Detects repeated messages and takes action.
Link Filter: Blocks messages with links (mods are exempt).
Max Warnings: How many warnings before a mute.
Mute Duration: How long mutes last in minutes.
Mod Log: Logs all moderation actions to a channel and the web dashboard.
Level System: Tracks member message activity for XP and levels.

Moderation

What slash commands are available?
/kick - Kick a member
/ban - Ban a member
/timeout - Timeout a member
/clear - Bulk delete messages
/slowmode - Set channel slowmode
/setup-server - Build the full server
/reset-server - Remove bot-created content
/help - Show command list
/info - Server info
/stats - Server statistics
/nexus - About Nexus
/link - Get your Nexus link
How does the warning system work?
When auto-moderation catches a violation (spam, blocked links), it issues a warning. After the configured number of warnings (default 3), the user is muted for the configured duration (default 10 minutes). Warnings reset after a mute.

Welcome System

How do I customize the welcome message?
Go to your server settings on the dashboard and edit the Welcome Message field. You can use these placeholders:
{user} - mentions the new member
{server} - the server name
{count} - current member count
Can I change which channel gets welcome messages?
The bot posts welcome messages in the channel named #welcome. If your server was set up with /setup-server, this channel already exists. You can rename it, but the bot looks for a channel called "welcome" specifically.

Logging

What gets logged?
Member joins and leaves, bans and unbans, kicks, timeouts, message deletions, message edits, bulk message deletions, role changes, nickname changes, and voice channel activity (join, leave, move). All of these are sent to the #mod-log channel and saved to the web dashboard.
Where can I view logs?
Two places: in the #mod-log channel in your Discord server, and on the web dashboard under your server's Mod Logs tab. The web dashboard lets you filter by action type and paginate through history.

Nexus Integration

What is Nexus?
Nexus is a YouTube bot and analytics platform. It provides live chat moderation, channel statistics, video management, and a dashboard for YouTube creators. The bot dashboard and the Nexus platform share the same account system, so you can link them together.
How do I link my Nexus account?
First, sign in to your Nexus account at nexusbeta.vercel.app in the same browser. Then go to your Account page here and click Link Account. Both sessions need to be active in the same browser for it to work.

Troubleshooting

The bot isn't responding to commands
Make sure the bot has the correct permissions in the channel you're using. Slash commands require the bot to have the Use Application Commands permission. If commands don't show up, try kicking and re-inviting the bot, then wait a few minutes for commands to sync.
I can't see my server on the dashboard
You need Manage Server permission or to be the server owner. Try logging out and back in to refresh your Discord permissions. If the issue persists, check your Discord role permissions.
Settings aren't saving
Make sure you click Save Settings at the bottom of the page. If you get an error, try refreshing the page and saving again. If the problem continues, contact me.
