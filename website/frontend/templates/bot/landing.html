What it actually does

Server Builder

Run /setup-server and get a full server layout. Admin, mod, and member roles. Text and voice channels sorted into categories. Hidden staff channels. AFK channel. Rules, FAQ, and role picker posts.

Auto Moderation

Set warning thresholds and mute durations. The bot catches spam and blocked links, warns the user, and mutes them after too many strikes. Mods are exempt by default.

Mod Logs

Kicks, bans, timeouts, message deletes, role changes, nickname edits, voice activity. Logged to a #mod-log channel in Discord and to the web dashboard where you can filter and search.

Welcome System

New member joins, bot posts a welcome message in #welcome. You pick the text and use placeholders like {user} and {server}. Pair it with auto-role to give everyone a default role on join.

Web Dashboard

Every toggle and setting lives here. Pick your server, flip switches, set values, save. No slash commands needed for configuration. View your mod logs from the same place.

Nexus Integration

Already use Nexus for YouTube? Link your accounts and manage both platforms from one place. Your Discord server and YouTube channel data stay connected.
